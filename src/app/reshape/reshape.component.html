ReSHAPE
The Ecological Restoration Institutes (SWERI) received congressional funding to visualize activities reduce the timber, brush, and other materials that fuel wildfires through a program called ReSHAPE.
To accomplish this, SWERI contracted with Innovate! Inc. to develop a web application with an interactive map viewer called TWIG.
My role in this project centered around research, design, and development of an intuitive and appealing UI/UX for the application, entailing the following functions:
  1. Creating personas, user stories, and functional requirements
  2. Designing interactive wireframes using design patterns for geospatial applications
  3. Building a theme and customizing components to complement the project's brand guide
  4. Developing code with team members using an Agile methodology
